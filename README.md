# Laboratório de Ideias

Este repositório é a memória permanente dos projetos.

## Estrutura

- `INDEX.md`: índice dos projetos.
- `PROJECT_TEMPLATE.md`: modelo para novos projetos.
- `.lab/`: infraestrutura operacional do Laboratório.

A pasta `.lab` contém as regras de colaboração para qualquer IA ou colaborador humano.
